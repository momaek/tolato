# ToLaTo Logo Assets

Files:

- `tolato-logo-horizontal-light.svg`
- `tolato-logo-horizontal-dark.svg`
- `tolato-logo-icon-light.svg`
- `tolato-logo-icon-dark.svg`
- `tolato-logo-mono-black.svg`
- `tolato-logo-mono-white.svg`

Recommended usage:

- Use horizontal logos for headers, docs, landing pages, and product chrome.
- Use icon logos for app icons, favicons, sidebar branding, and compact surfaces.
- Keep the yellow accent only on the origin dot in color versions.
- Prefer SVG as the source of truth, then rasterize to PNG as needed.
